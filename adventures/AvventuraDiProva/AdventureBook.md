# AdventureBook.md — Istruzioni per Kiro

Questa avventura segue la struttura standard definita in `adventures/AdventureTemplate/AdventureBook.md`.

## Note specifiche

- È il template di riferimento del progetto — one-shot minimale, tutti i tipi di file presenti
- I PG (Mario, Lucia, Barbara) sono predefiniti e inclusi in `characters/` come esempio; in un'avventura normale le schede PG non fanno parte del repository
- Cattivone è l'antagonista principale con scheda completa; Panza Grande e Rubizzo sono PNG secondari con scheda semplificata
- Il finale è volutamente aperto: Cattivone sparisce, aggancio per avventure future
